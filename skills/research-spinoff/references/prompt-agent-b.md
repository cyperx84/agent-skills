# Agent B — Creative, AI-Native & Non-Obvious

Research non-obvious, creative product ideas at the intersection of **{{PROJECT}}** and emerging trends.

{{CONTEXT}}

## Research Areas

### 1. AI-native products that don't exist yet
- What if AI watched user behavior and proactively applied the core concept?
- Real-time assistants, ambient intelligence, copilot patterns

### 2. Education / gamification
- Duolingo-style learning, case studies, simulation, persona chatbots

### 3. Team / enterprise
- Decision audit trails, compliance, group workflows, retrospectives

### 4. Consumer / lifestyle
- Quick-decision apps, personal life tools, relationship/career/finance

### 5. Developer tools
- IDE extensions, CI integrations, code review augmentation, ADR-style tools

## Output Format

For EACH idea (aim for 15-20):

| Field | Content |
|-------|---------|
| **Name** | Suggested product name |
| **Pitch** | One-line description |
| **Audience** | Who it's for |
| **Differs from source** | How it's different from {{PROJECT}} |
| **Revenue** | Free / freemium / paid + price point |
| **Build time** | Weekend / 2 weeks / month / quarter |
| **Exists?** | Closest competitors (name + URL if found) |

Be creative. Look for gaps nobody is filling. The weird ideas are often the best ones.
