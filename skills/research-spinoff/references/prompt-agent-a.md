# Agent A — Surfaces, Audiences & Domains

Research what standalone apps, tools, CLIs, or products could be built from the same foundation as **{{PROJECT}}**.

{{CONTEXT}}

## Research Areas

### 1. Different surfaces / form factors
- Mobile app, web app / SaaS, browser extension, Slack/Discord bot
- Obsidian/Notion plugin, VS Code extension, API service, newsletter

### 2. Different audiences
- Founders, PMs, investors, students, coaches, writers, managers, engineers

### 3. Different problem domains
- Look at adjacent domains the core idea could serve
- What workflows in other industries use similar patterns?

### 4. Monetization angles
- Which could be paid products?
- Which have existing markets with paying customers?

## Output Format

For EACH idea (aim for 15-20):

| Field | Content |
|-------|---------|
| **Name** | Suggested product name |
| **Pitch** | One-line description |
| **Audience** | Who it's for |
| **Differs from source** | How it's different from {{PROJECT}} |
| **Revenue** | Free / freemium / paid + price point |
| **Build time** | Weekend / 2 weeks / month / quarter |
| **Exists?** | Closest competitors (name + URL if found) |
